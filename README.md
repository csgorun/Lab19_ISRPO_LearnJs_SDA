## Краткое описание работы
Целью данной лабораторной работы было углубленное изучение языка программирования JavaScript через сравнение с C#. В ходе работы были реализованы базовые конструкции языка: циклы, функции, массивы и объекты, а также настроено окружение для разработки и версия контроля.

## Описание (что изучили)
В ходе выполнения работы были изучены и применены на практике следующие темы:
- **Циклы:** `for`, `while`, `do...while`, операторы `break` и `continue`, вложенные циклы.
- **Функции:** объявление, параметры, значения по умолчанию, стрелочные функции, замыкания.
- **Массивы:** создание, доступ к элементам, методы `push`/`pop`, перебор (`for...of`), поиск (`indexOf`, `includes`).
- **Объекты:** литералы объектов, доступ к свойствам, методы объекта, перебор свойств (`for...in`), вложенные структуры.
- **Инструменты:** работа с Git, GitHub, VS Code, запуск кода в браузере и Node.js.
